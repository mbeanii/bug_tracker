The Python Sqlite3 library provides simple interface with a Sqlite3.
I chose it due to its ease of implementation, python compatibility, and abundant documentation.
The database directory contains bug_tracker.db, tests.py, and titanic_db.py. The latter contains
logic for initialization of and interfacing with the database. Instructions for reinitializing
the db are available in README.md.
